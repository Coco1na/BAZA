**Теория**📖

**Физическая величина** - это количественная характеристика тела и вещества. У нее есть единица измерения, буквенное обозначение и название. Например: масса, длина волны или внутренняя энергия.

**Единица физическое величины** -  это фиксированное значение величины, которое используется для её количественного выражения. Примером такой единицы является нанометр, герц или секунды.

**Физическое явление** - это наблюдаемое изменение в природе, происходящее без изменения состава вещества. Примером такого явления является взаимодействие молекул, радуга, гром. 

**Приборы**🧭 

- **Спидометр** — измерительный прибор для определения модуля мгновенной скорости движения. 
- **Вольтметр** — измерительный прибор для определения разности электрических потенциалов, напряжения в электрических цепях.
- **Амперметр** — измерительный прибор для измерения силы тока. 
- **Омметр** — измерительный прибор для определения электрических сопротивлений. 
- **Ваттметр** — измерительный прибор, предназначенный для определения мощности электрического тока. 
- **Реостат** — прибор для регулирования силы тока и его напряжения.
- **Манометр** — прибор для измерения давления жидкости или газа в замкнутом пространстве. 
- **Электрометр** — измерительный прибор для обнаружения электрических зарядов. 
- **Акселерометр** — прибор, измеряющий проекцию кажущегося ускорения.

- **Барометр-анероид** — прибор для измерения атмосферного давления, в отличие от ртутного барометра, действующий без помощи жидкости. 
- **Высотомер** — измерительный прибор, предназначенный для измерения высоты. 
- **Компас** — измерительный прибор для обнаружения взаимодействия постоянных магнитов.

- **Секундомер** — измерительный прибор, который предназначается для измерения времени. 
- **Гигрометр** — измерительный прибор, предназначенный для определения влажности воздуха и различных газов. 
- **Психрометр** — прибор, который предназначается для измерения температуры воздуха и его влажности. 
- **Термометр** — измерительный прибор для измерения температуры различных тел и сред.

- **Мензурка** — сосуд с делениями для отмеривания жидкостей.  -
