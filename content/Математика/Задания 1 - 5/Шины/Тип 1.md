Многие считают шины самым страшным типом заданий, давай разберемся с ними и поймем как это легко. Погнали🛞

![[img58.png]]

Автомобильное колесо, как правило, представляет из себя металлический диск с установленной на него резиновой шиной. Диаметр диска совпадает с диаметром внутреннего отверстия в шине. 

Для маркировки автомобильных шин применяется единая система обозначений. Например, 195/65 R15 (рис. 1). Первое число (число 195 в приведённом примере) обозначает ширину шины в миллиметрах (параметр B на рисунке 2). Второе число (число 65 в приведённом примере) — процентное отношение высоты боковины (параметр H на рисунке 2) к ширине шины, то есть **100 * H/B**

Последующая буква обозначает тип конструкции шины. В данном примере буква R означает, что шина радиальная, то есть нити каркаса в боковине шины расположены вдоль радиусов колеса. На всех легковых автомобилях применяются шины радиальной конструкции. 

За обозначением типа конструкции шины идёт число, указывающее диаметр диска колеса d в дюймах (в одном дюйме 25,4 мм). Таким образом, общий диаметр колеса D легко найти, зная диаметр диска и высоту боковины. 

Возможны дополнительные маркировки, обозначающие допустимую нагрузку на шину, сезонность использования, тип дорожного покрытия и другие параметры. Завод производит легковые автомобили определённой модели и устанавливает на них колёса с шинами маркировки 165/70 R13.

### Задание 2

На сколько миллиметров радиус колеса с шиной маркировки 205/55 R14 больше, чем радиус колеса с шиной маркировки 165/65 R14?

Для начала вспомним теорию

> [!example] Теория
> Радиус окружности равен половине ее диаметра

Чтобы найти радиус колес, нужно сначала найти диаметры колес

![[img61.png]]

Диаметр колеса равен

$$D = H + d +H$$

Где Н - высота боковины, а d - диаметр диска колеса

Прочитаем еще раз условие

> [!quote] Условие
> Для маркировки автомобильных шин применяется единая система обозначений. Например, 195/65 R15 (рис. 1). Первое число (число 195 в приведённом примере) обозначает ширину шины в миллиметрах (параметр B). Второе число (число 65 в приведённом примере) — процентное отношение высоты боковины (параметр H) к ширине шины, то есть **100 * H/B**

Изучив условие, мы сможем найти высоты боковины, при помощи такой формулы

$$65 = 100 * \frac{H}{B}$$

65 - это второе число из маркировки шины (195/65 R15)

100 - необходимо для перевода в проценты

Н - высота боковины в миллиметрах

B - ширина шины в миллиметрах 

Выразим из формулы высоту боковины

$$H = \frac{65*B}{100}$$

Число 65 будем менять в соответствии с маркировкой шины

**Шаг 1 - Найдем высоту боковины для каждой шины** 

Шина маркировки 205/55 R14

$$H = \frac{55*205}{100} = 112,75 \text { мм}$$

Шина маркировки 165/65 R14

$$H = \frac{65*165}{100} = 107,25 \text { мм}$$

Теперь найдем диаметр диска колеса

> [!quote] Условие
> За обозначением типа конструкции шины идёт число, указывающее диаметр диска колеса d в дюймах (в одном дюйме 25,4 мм) - 195/65 **R15**. 

Чтобы найти диаметр диска колеса нужно умножить число стоящее после R на 25,4. 

**Шаг 2 - Найдем диаметры дисков** 

Шина маркировки 205/55 R14

$$d = 14 * 25,4 = 355,6 \text { мм}$$

Шина маркировки 165/65 R14

$$d = 14 * 25,4 = 355,6 \text { мм}$$


**Шаг 3 - Найдем диаметры и радиус колес**

Шина маркировки 205/55 R14

$$D = H + d +H = 2H +d = 2 * 112,75 + 355,6= 581,1 \text{ мм}$$

$$R = \frac {D}{2}=\frac {581,1}{2} = 290,55 \text{ мм}$$

Шина маркировки 165/65 R14

$$D = H + d +H = 2H +d = 2 * 107,25 + 355,6= 570,1 \text{ мм}$$

$$R = \frac {D}{2}=\frac {570,1}{2} = 285,05 \text{ мм}$$

**Шаг 4 - Найдем разницу между радиусами шин**

Просто вычтем один радиус из другого

$$\text{Разница радиусов} = R1 - R2 =290,55 - 285,05 = 5,5  \text{ мм}$$

В ответ запишем число: **5,5**

### Задание 3

На сколько миллиметров увеличится диаметр колеса, если заменить колёса, установленные на заводе, колёсами с шинами маркировки 195/50 R15?

Из условие прочитаем, какие шины производят на заводе

> [!quote] Условие
> Завод производит легковые автомобили определённой модели и устанавливает на них колёса с шинами маркировки 165/70 R13.

Найдем диметры колес маркировки 165/70 R13, 195/50 R15 и сравним их

**Шаг 1 - Найдем высоту боковины для каждой шины** 

Шина маркировки 165/70 R13

$$H = \frac{70*165}{100} = 115,5 \text { мм}$$

Шина маркировки 195/50 R15

$$H = \frac{50*195}{100} = 97,5 \text { мм}$$

**Шаг 2 - Найдем диаметры дисков** 

Шина маркировки 165/70 R13

$$d = 13 * 25,4 = 330,2 \text { мм}$$

Шина маркировки 195/50 R15

$$d = 15 * 25,4 = 381 \text { мм}$$

**Шаг 3 - Найдем диаметры колес**

Шина маркировки 165/70 R13

$$D = H + d +H = 2H +d = 2 * 115,5 + 330,2= 561,2 \text{ мм}$$

Шина маркировки 195/50 R15

$$D = H + d +H = 2H +d = 2 * 97,5 + 381= 576 \text{ мм}$$

Осталось посчитать разницу в диаметре новых шин и шин с завода

$$\text{Разница в диаметре} = D1 - D2 = 576 - 561,2 = 14,8 \text{ мм}$$

В ответ запишем число: **14,8**

### Задание 5

На сколько процентов увеличится пробег автомобиля при одном обороте колеса, если заменить колёса, установленные на заводе, колёсами с шинами маркировки 175/60 R14? Результат округлите до десятых.

$$C = 2πR = 2D$$

R - радиус окружности

D - диаметр окружности

В этом задании нам нужно будет найти длины окружностей разных колес и сказать на сколько процентов изменится пробег, если поменять заводские шины на новые. Для начала найдем диаметр шин

**Шаг 1 - Найдем высоту боковины для шин**

Шина маркировки 165/70 R13

$$H = \frac{70*165}{100} = 115,5 \text { мм}$$

Шина маркировки 175/60 R14

$$H = \frac{60*175}{100} = 105 \text { мм}$$

**Шаг 2 - Найдем диаметры дисков** 

Шина маркировки 165/70 R13

$$d = 13 * 25,4 = 330,2 \text { мм}$$

Шина маркировки 175/60 R14

$$d = 14 * 25,4 = 355,6 \text { мм}$$

**Шаг 3 - Найдем диаметры колес**

Шина маркировки 165/70 R13

$$D = H + d +H = 2H +d = 2 * 115,5 + 330,2= 561,2 \text{ мм}$$

Шина маркировки 175/60 R14

$$D = H + d +H = 2H +d = 2 * 105,5 + 355,6= 565,6 \text{ мм}$$

**Шаг 3 - составим пропорцию**

![[img62.png]]

Решив пропорцию найдем, что пробег шины с завода - это 100%, а шины маркировки 175/60 R14 - 100,78%. Значит пробег автомобиля при одном обороте колеса увеличится на

Результат нужно округлить до десятых.
