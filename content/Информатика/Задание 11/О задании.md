### Архивирование данных

![|550](img102.jpg)

Архивирование применяется для освобождения места на диске, упрощения поиска информации, отправки больших объемов информации.

Существует множество программ для архивации данных WinRAR, 7-Zip, Bandizip, Keka и другие, но суть их работы очень похожа, давай разберем как разархивировать данные. 

На ОГЭ на вашем рабочем столе будут находится архивы для заданий из второй части, выглядеть архивы будут так:

![[img103.png]]

Для того чтобы распаковать архив нужно нажать на него правой кнопкой мыши (ПКМ), выбрать вкладку действия:

![[img104.jpg]]


Далее нажимаем «Распаковать в...», выбираем «Рабочий стол» и нажимаем кнопку «Распаковать»:

![[img105.jpg]]

После этого на рабочем столе появится распакованная папка с которой можно работать:

![[img106.jpg]]

Попробуем сделать тоже самое на Windows, на рабочем столе также лежит архив:

![[img107.png]]

Нажимаем два раза левой кнопкой мыши и откроется окно программы архиватора:

![[img108.png]]

Нажимаем на папку, которую нужно разархивировать и нажимаем на кнопку извлечь (синенькая в верхнем левом углу). В открывшемся окне выбираем место для сохранения  «Рабочий стол» и нажимаем кнопку «ОК»:

![[img109.png]]

После этого на рабочем столе появится распакованная папка с которой можно работать:

![[img110.png]]

Теперь давай попробуем использовать средства поиска операционных систем.

### Средства поиска

В 11-ом задании необходимо искать данные в тексте. У вас будет несколько файлов разного расширения с текстовым содержанием. Для того чтобы найти данные внутри файла в Windows 10 нужно нажать комбинацию клавиш Ctrl +F1, на Windows 11 нажать три точки в проводнике и выбрать вкладку «Параметры»

![[img111.png]]

В открывшейся вкладке выбираем окно поиска и ставим настройки как на рисунке ниже (в Windows 10 ставим галочку рядом с «Поиск по содержимому»)

![|400][img112.png]

Теперь поиск настроен и в верхнем левом углу проводника в строку поиска можно вводить слово или словосочетание, которое нужно найти:

![|400][img113.png]

В Linux это сделать также просто. Для поиска содержимого в файле нажимаем на значок 🔍

![|400][img114.png]

В открывшемся окне выбираем вкладку «Содержимое», вводим искомое слово и нажимаем кнопку «Найти»

![|400][img115.png]

### Поиск внутри файлов

>[!success] Подсказка
>
>**Для поиска содержимого внутри файла почти любого расширения используется комбинация клавиш Ctrl + F**

Например у нас есть текстовый файлик

![|500][img116.png]

Откроем его и нажмем Ctrl + F и для нас откроется окно поиска

![|400][img117.png]

Все найденные слова будут выделяться желтым цветом
