Существует три главных агрегатных состояния: твердое, жидкое, газообразное. Вещество переходит из одного состояния в другое из-за нескольких явлений.

|          Название           |   Формула    | Смайлик |
| :-------------------------: | :----------: | :-----: |
|    Нагревание/охлаждение    | $$Q = cmΔt$$ |   🔥    |
|  Плавление/кристаллизация   |  $$Q = λm$$  |   🌋    |
| Парообразование/конденсация |  $$Q = Lm$$  |   💨    |

**c**  - удельная теплоемкость вещества. Физическая величина, равная количеству теплоты, которое необходимо сообщить 1 кг вещества для нагревания его на 1 °С

**λ** - удельная теплота плавления. Количество теплоты, которое необходимо сообщить кристаллическому телу массой 1 кг, чтобы при температуре плавления полностью перевести его в жидкое состояние. 

**L** - удельная теплота парообразования (теплота испарения). Количество теплоты, которое нужно подвести к 1 кг жидкости, взятой при температуре кипения, чтобы полностью превратить её в пар.
